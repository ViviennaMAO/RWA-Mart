# 产品需求文档 (PRD)：Luffa RWA Mart 小程序

## 1. 文档概述

- **产品名称**：RWA Mart (Luffa Mini-Program)
    
- **版本**：V1.0.0
    
- **状态**：初稿
    
- **目标**：为 Luffa 用户提供一站式的现实世界资产（RWA）链上数据监控、分析及投资入口。
    

---

## 2. 产品定位

利用 Luffa 的多链钱包能力和社交属性，将 RWA.xyz 的复杂数据转化为移动端友好的交互界面，帮助用户快速捕捉私人信贷、代币化国债及房地产等赛道的投资机会。

---

## 3. 用户功能需求

### 3.1 首页：市场概览 (Market Overview)

- **全网 TVL 汇总**：显示 RWA 赛道总锁定价值，以及 24 小时涨跌幅。![[截屏2026-01-20 23.34.52.png]]

- **GLOBAL MACRO data**：FED FUND RATE/CPI/VIX/MOVE/US 10Y TIELD（右滑显示更多）

![[截屏2026-01-20 23.35.14.png]]
- **Total RWA Value**：US Treasury Debt/Commodities/Private Credit/Institutional Alternative Funds/Public Equity/non-US Government Debt/Private Equity![[截屏2026-01-20 23.35.54.png]]
    
- **热门协议滚动**：展示当前 APY 最高或 TVL 增长最快的 Top 3 协议，可切换tab选择，右滑查看更多数据。
    ![[截屏2026-01-20 23.36.22.png]]

### 3.2 资产列表与筛选 (Discovery)![[截屏2026-01-20 23.37.53.png]]

- **多维度排序**：支持按 TVL、APY、所属公链、信用等级进行排序。
    
- **分类标签**：用户可一键切换查看“美债”或“私人信贷”等细分领域。
    
- **实时数据点**：显示每个协议的代币价格（如有）和当前活跃贷款额。
    
![[截屏2026-01-20 23.38.17.png]]
### 3.3 协议详情页 (Protocol Details)![[截屏2026-01-20 23.38.17.png]]

- **资产构成**：详细列出该协议背后的抵押物信息。
    
- **收益分析**：展示历史收益率曲线。
    
- **快捷交互**：点击“去投资”可跳转至该协议官网或通过 Luffa 内置浏览器打开。

## 4. 技术需求 (Luffa SDK 集成)

| **功能模块** | **技术实现要点**                                                                                                                |
| -------- | ------------------------------------------------------------------------------------------------------------------------- |
| **容器适配** | 使用 H5 响应式布局，适配 Luffa `superbox-mini-sdk` 环境。                                                                              |
| **钱包连接** | 调用 `luffa.getAccount()` 获取当前登录地址。api：<br><br>https://luffa.im/SuperBox/docs/hk/API/customizeAPI.html#t-locallang-api-3444 |
| **链上交互** | 调用 `luffa.sendTransaction()` 支持在小程序内直接参与 RWA 质押。                                                                          |
| **分享功能** | 集成 `luffa.setShareInfo()`，支持用户将特定数据看板分享至 Luffa 群聊。                                                                        |
| **数据源**  | 初始阶段通过 `fetch` 接入 RWA.xyz 或 Dune Analytics API；二期考虑接入 Indexer。                                                            |
